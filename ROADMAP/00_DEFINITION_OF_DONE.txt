================================================================================
DEFINITION OF DONE
================================================================================
Project: MT5 Automated Forex Trading Bot

A phase is NOT complete simply because code was generated.
A task is NOT complete simply because the agent reported success.

================================================================================
TASK-LEVEL DEFINITION OF DONE
================================================================================

A task is COMPLETE only when ALL of the following are true:

  □ Implementation exists (the code has been written)
  □ Code is syntactically valid (no Python syntax errors)
  □ All imports resolve correctly
  □ The module can be imported: python -c "import app.[module]"
  □ All specified tests pass: pytest tests/[relevant_test_file].py
  □ Edge cases specified in the task are tested
  □ No critical errors remain (warnings are acceptable if documented)
  □ Scope was respected (no unrelated files were modified)
  □ No hardcoded secrets or credentials
  □ Code follows patterns in CODE_STANDARDS.md
  □ Acceptance criteria from the task file are satisfied
  □ Task marked [✓] in ROADMAP/00_PROJECT_STATUS.txt

================================================================================
PHASE-LEVEL DEFINITION OF DONE
================================================================================

A phase is COMPLETE only when ALL tasks in the phase are [✓] DONE, AND:

  □ All phase tests pass (full test run for the phase scope)
  □ Integration with previously completed phases works
  □ No regressions introduced in previously working code
  □ Phase acceptance criteria are satisfied
  □ All files expected to be created exist
  □ No files were unexpectedly modified outside the phase scope
  □ Phase marked [✓] in ROADMAP/00_PROJECT_STATUS.txt
  □ Checkpoint created
  □ Known limitations documented

WHAT "DONE" IS NOT
================================================================================

"Done" is NOT:
  ✗ "The agent says it's complete"
  ✗ "The code looks reasonable"
  ✗ "I didn't get an error"
  ✗ "It worked once in manual testing"
  ✗ "The tests I wrote pass"

"Done" IS:
  ✓ All specified tests pass
  ✓ Edge cases tested
  ✓ Code reviewed by a human (you)
  ✓ Acceptance criteria verified
  ✓ No scope violations
